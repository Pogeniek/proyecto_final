# Proyecto de Automatización QA - Marcos Vargas

> Suite de pruebas automatizadas para aplicación web | Python + Selenium + Pytest

---

## 📋 Descripción

Proyecto de automatización de pruebas funcionales para una aplicación web, implementado con:

- **Python** como lenguaje principal
- **Selenium WebDriver** para interacción con navegadores
- **Pytest** como framework de pruebas
- **Behave** para pruebas BDD (Behavior Driven Development)
- **Requests** para pruebas de API

**Objetivo:** Garantizar la calidad del sistema mediante pruebas automatizadas de regresión, funcionales y de integración.

---

## 🛠️ Tecnologías

| Herramienta | Versión | Propósito |
|-------------|---------|-----------|
| Python | 3.8+ | Lenguaje de programación |
| Selenium | 4.36.0 | Automatización de navegadores |
| Pytest | 8.4.2 | Framework de pruebas |

| Pytest-Check | 2.8.0 | Validaciones múltiples |
| Behave | 1.3.3 | Pruebas BDD (Gherkin) |
| Requests | 2.32.5 | Pruebas de API |
| WebDriver-Manager | 4.0.2 | Gestión automática de drivers |
